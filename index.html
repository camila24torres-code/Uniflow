Tu asistente universitario en la nube ☁️
1 Crea tu proyecto en Firebase
Ve a console.firebase.google.com → "Añadir proyecto" → ponle cualquier nombre → desactiva Google Analytics → clic en "Crear proyecto".
2 Crea la base de datos Realtime
En el menú izquierdo: Compilación → Realtime Database → Crear base de datos → selecciona ubicación us-central1 → elige "Modo de prueba" → Habilitar.
3 Registra tu app web
En el menú: Configuración del proyecto ⚙️ → Tus apps → </> (web) → ponle nombre → Registrar app. Copia el valor de databaseURL que aparece.
